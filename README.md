# booo

CLI tool for bootstrapping development environments.

## What is booo?

**B**un, **O**penCode, **O**penSpec and **O**bsidian

A CLI tool that sets up your project with OpenSpec, OpenCode skills, and Obsidian configuration.

## Installation

```bash
brew install gouz/tools/booo
```

## Usage

```bash
booo setup
```

The setup command will:
1. Initialize OpenSpec with OpenCode tools
2. Prompt to create a devcontainer configuration
3. Install Obsidian rules for your project
